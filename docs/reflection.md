# Reflection

## What I Built

I developed a full-stack SupportFlow application consisting of:

- React frontend
- Express backend
- MongoDB database
- JWT authentication
- Ticket Management
- Team Management
- Reports Dashboard

## AI Workflow

AI was used throughout the development lifecycle for:

- Requirement clarification
- Planning
- Debugging
- Code review
- Validation
- Documentation

Generated suggestions were reviewed before being accepted and integrated.

## Challenges

The most significant challenges included:

- Authentication persistence
- MongoDB connectivity
- Frontend/backend integration
- Ticket CRUD integration
- Maintaining consistent project architecture

Each issue was investigated before applying a solution.

## What Went Well

- Incremental development
- Consistent architecture
- Reusable components
- Successful production review
- Complete end-to-end functionality

## What I Would Improve

With additional time I would add:

- Role-based authorization
- Automated tests
- Activity logs
- Notifications
- Docker Compose deployment

## Key Learnings

This project strengthened my understanding of integrating AI into software development while maintaining ownership of implementation decisions and validating generated code before adoption.
